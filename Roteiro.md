Para ser um humano notável, é necessário habilidades fisicas, eu considero as mais importantes: corrida, luta. Então decidi criar um jogo em que o objetivo do jogador é aprimorar essas duas habilidades
em que o jogador inicia o jogo podendo escolher um visual e nome, ele começa com nível de habilidade baixo. No menu, ele poderá escolher jogar um jogo dentre os dois.  A cada vitoria ou conquista seu nivel aumentar com alguns pontos. 

Quando o jogador ganhar ou perder voltará ao LOB, onde pode acessar a loja e comprar visuais e habilidades. 
Também terá um caminho de pontos, em que a medida que avança ele poderá desbloquear perfis. Os que pensei até agora: Iniciante, profissional, elite,AURA INFINITA.Esses nomes devo alterar :)

No minijogo de luta será o jogador contra um bot, em que ele pode usar ataques e poderes contra o inimigo{
    Devera ser um jo 2D. Vou usar o java scrpit para detectar a posição do personagem e do ataque, se estiverem na mesm aposição quer dizer que o jogador foi atingido, perdendo vida(usarei um contador para aber quando ele perdeu).
}
  
No de corrida sera tipo o jogo do dino do google, mas com uma pessoa correndo e saltando barreiras(cacto, lama,...){
  Pretendo usar @keyframes para a movimentação dos obstáculos.
  Um java script para detectar se o jogador esta na mesma posição que o obstáculo, se sim: eliminado!
  E um java script para contar tempo e pontos(talvez uma variavel: contador++, a cada ponto);
}

obs: tentarei colocar musica e uma versão para cegos com um sistema de sons quando a barreira tiver perto.

Loja: usarei java script para fazer um sistema de saldo.

Enfim, nesse jogo o objetivo é se aprimorar como um ser humano.
